# Security Policy

This repository is a private, internal API (`velytics-backend`) and does not
maintain a public version-support matrix.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by
emailing **stevenrq8@gmail.com** instead of opening a public GitHub issue.

Please include:

- A description of the vulnerability and its potential impact.
- Steps to reproduce (a proof of concept, if possible).
- Any relevant logs, request/response samples, or affected endpoints.

You should expect an initial response within **5 business days**. We ask
that you give us a reasonable amount of time to investigate and address the
issue before any public disclosure.

## Scope

This policy covers the `velytics-backend` API. Dependency vulnerabilities are
also tracked automatically via Dependabot (`.github/dependabot.yml`) and
`npm audit` in CI.
